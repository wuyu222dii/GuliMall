unicodeQuote

unicodeIt is the most original way to apply fonts on the web page. Its characteristics are:

  • Best compatibility, supportie6+, and all modern browsers.
  • Supports dynamic adjustment of icon size, color, etc. based on font.
  • But because it is a font, it does not support multiple colors. Only single-color icons in the platform can be used. Even if there are multi-color icons in the project, the colors will be automatically removed.

Note: new versioniconfontSupports multi-color icons, these multi-color icons areunicodeIt cannot be used in mode. It is recommended to use it if necessary.symbolReference method

unicodeThe usage steps are as follows:

Step 1: Copy the generated files below the projectfont-face

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.eot');
  src: url('iconfont.eot?#iefix') format('embedded-opentype'),
  url('iconfont.woff') format('woff'),
  url('iconfont.ttf') format('truetype'),
  url('iconfont.svg#iconfont') format('svg');
}

Step 2: Define usageiconfontstyle

.iconfont{
  font-family:"iconfont" !important;
  font-size:16px;font-style:normal;
  -webkit-font-smoothing: antialiased;
  -webkit-text-stroke-width: 0.2px;
  -moz-osx-font-smoothing: grayscale;
}

Step 3: Select the corresponding icon and obtain the font encoding, and apply it to the page

<i class="iconfont">&#x33;</i>

"iconfont"It’s under your projectfont-family. Can be viewed by editing the project, the default is"iconfont".

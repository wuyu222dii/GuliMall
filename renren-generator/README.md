**Project Description** 
- renren-generator is a code generator from Renren Open Source. It generates entity, xml, dao, service, html, js, and sql online, reducing 70%+ of boilerplate work.

 **Local Setup**
- Clone the repo with git
- Edit application.yml with MySQL credentials and database name
- Run RenrenApplication.java in Eclipse or IDEA
- App URL: http://localhost
